# n8n Workflow Architect

This skill enables bionicbot to design complex automation flows using the n8n visual engine, connecting 500+ apps locally or via cloud.

## Instructions
1. Map out the trigger (Webhook, Schedule, or App Event).
2. Design the logical path using 'If' and 'Switch' nodes.
3. Integrate 'Code' nodes for custom Python/JS data transformation.
4. Set up 'Error Trigger' flows to automatically notify the user on failure.

## Examples
- "Design an n8n flow that scrapes LinkedIn and sends leads to our Sovereign Vault."
