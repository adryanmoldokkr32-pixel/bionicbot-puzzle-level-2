# Recursive Code Refactorer

This skill enables bionicbot to act as a world-class software architect, rewriting entire systems to reduce complexity and increase speed.

## Instructions
1. Scan the project for 'Big O' inefficiencies.
2. Identify patterns that violate DRY (Don't Repeat Yourself) principles.
3. Automatically propose a modular structure for monolithic files.
4. Ensure 100% test coverage before and after refactoring.

## Examples
- "Refactor our 'Sovereign-OS' backend to handle 10x more concurrent users."
